# Financeiro Seguro

Projeto simples de controle financeiro pessoal criado com React + Vite.

Como rodar localmente:

```powershell
cd "C:\Users\pierr\OneDrive\Documentos\Nova pasta\projeto"
npm install
npm run dev
```

Como gerar build para produção:

```powershell
npm run build
```

Dicas para deploy:
- No Vercel, use `npm run build` como Build Command e `dist` como Output Directory.
- Ou publique este repositório no GitHub e conecte ao Vercel para deploy automático.
